# Windows
MSI setup download link - https://awscli.amazonaws.com/AWSCLIV2.msi

Run msi setup

*msiexec.exe /i https://awscli.amazonaws.com/AWSCLIV2.msi*

Check aws version in new command prompt

*aws --version*

Configure aws cli
*aws configure*
